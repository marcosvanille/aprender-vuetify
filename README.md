# Aprender Vuetify

## Criação
``` sh
vue create aprender-vuetify 
cd aprender-vuetify      
npm run serve 
```
acessar no navegador: http://localhost:8080/
``` sh
vue add vuetify 
npm run serve
```
acessar no navegador: http://localhost:8080/

Agora temos o inicio dos trabalhos prontos.


## Instalar Projeto.
```sh
git clone https://github.com/wouerner/aprender-vuetify.git
cd aprender-vuetify
npm install
npm run serve
```
